# Tisptari

## Tuner CSV

Output for 2700 runs with different parameters that we used for tuning our algorithm.

## Optional: Create Virtual Environment

Use a virtual environment to avoid conflicts with other Python packages. The exact command may vary depending on your operating system.

    python3 -m venv .venv
    source .venv/bin/activate

## Install Dependencies

    pip install -r requirements.txt

## Run Program

    python main.py
    python3 main.py
